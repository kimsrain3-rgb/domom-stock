# 두맘코튼 재고관리 웹앱

두맘코튼(DO.MOM Cotton) 23색 뜨개실 재고관리 앱

## 배포 URL
https://kimsrain3-rgb.github.io/domom-stock/

---

## 설정 순서 (최초 1회)

### 1단계 — 구글 시트 만들기
1. 구글 드라이브에서 새 스프레드시트 생성
2. 시트 이름을 다음 4개로 설정:
   - `재고현황`
   - `입출고기록`
   - `발주관리`
   - `사용자`

3. **재고현황** 시트 1행 헤더:
   `번호` | `영문명` | `한글명` | `현재 재고` | `안전 재고` | `총 입고` | `총 출고` | `상태`

4. **입출고기록** 시트 1행 헤더:
   `이력ID` | `날짜` | `컬러번호` | `컬러명` | `구분` | `수량(볼)` | `메모` | `담당자`

5. **사용자** 시트 1행 헤더:
   `ID` | `PW(해시)` | `솔트` | `이름` | `역할` | `생성일`

6. 재고현황 시트에 23색 데이터 입력 (안전 재고는 원하는 수량으로)

### 2단계 — Apps Script 설정
1. 구글 시트 메뉴 → **확장 프로그램 → Apps Script**
2. `gas/Code.gs` 파일 내용을 전체 복사 → Apps Script 편집기에 붙여넣기
3. **저장** → **배포 → 새 배포**
4. 유형: **웹 앱**, 액세스: **모든 사용자** 선택 후 배포
5. 배포 URL 복사 (앱 설정에서 사용)

### 3단계 — 첫 번째 관리자 계정 만들기
Apps Script 편집기에서 다음 함수를 직접 실행:
```javascript
createUser('admin', '원하는비밀번호', '꼼지파파', 'admin')
```

### 4단계 — 앱 설정
1. `https://kimsrain3-rgb.github.io/domom-stock/` 접속
2. GAS URL 입력창에 배포 URL 붙여넣기
3. 관리자로 로그인 후 설정 탭에서 Claude API 키 입력

### 5단계 — 자동 백업 설정
Apps Script 편집기에서:
- **트리거** → **트리거 추가**
- 함수: `dailyBackup`, 이벤트: 시간 기반, 새벽 1~2시
